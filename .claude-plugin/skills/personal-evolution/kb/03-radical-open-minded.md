# 3. Be Radically Open-Minded

## Two Barriers to Good Decisions

### 1. Ego Barrier
- Subliminal defense mechanism → hard to accept weakness/mistake
- From deepest needs/fears: being loved, survival, mattering
- Located in amygdala (temporal lobe) → processes emotions
- Craves praise, sees criticism as attack
- HL (higher-level) understands constructive criticism is good
- LL (lower-level) fights back anyway

### 2. Blind Spot Barrier
- View things in own way only
- [big picture VS details], [linear VS divergent] thinking
- Presume too much → occupied by telling others you're correct
- Miss: wonderful possibilities, dangerous threats, lifesaving criticism

### The Battle
```
Higher-Level (neocortex)     vs     Lower-Level (amygdala)
   ↓                                      ↓
Logic, reasoning              ←→      Emotions, instincts
Conscious awareness                   Hidden motives
"Why did I eat the cake?"            "Just eat it!"
```

### Fatal Flaw Pattern
- Ego + Blind Spots = prevent intelligent people from reaching potential
- Aristotle: "Tragedy = terrible outcome from person's fatal flaw"
- Could be wonderful if flaws were compensated

## 3.1 Practice Radical Open-Mindedness

### Definition
```
OM = genuine fear (self-view ≠ truth)
   + acknowledge blind spots exist
   + explore viewpoints/possibilities
   + higher-level thinking based on data
```

### Key Practices
| Practice | Description |
|----------|-------------|
| Unknown > "known" | Ask right questions, seek advice |
| Produce → Absorb | Listen before speaking |
| Suspend judgment | Encourage others to say you're wrong |
| Best answer anywhere | Not just your answer |
| Check "single-lens" | Am I only seeing my perspective? |

### Believability Criteria
Someone is believable when:
1. 3+ successful experiences in relevant field
2. Can explain cause-effect when probed

### Mode Selection
| Situation | Mode |
|-----------|------|
| Peer (equal believability) | Debate |
| Expert vs Student | Learn/Teach |
| Uncertainty | Ask questions |

## 3.2 Thoughtful Disagreement (TD)

### Motivation
- Genuine fear of being wrong
- Goal: find right view → action

### TD Rules
- Two higher-level minds communicate
- Skill: question > statement
- Manner: calm, equal, respect
- Winner = learns → changes quickly
- Loser = stubborn

### Confirmation Technique
Repeat other's view → confirmed → good

### Avoid
- Thinking in your head alone
- Wasting time in unproductive discussion
- Spending time on non-believable people

## 3.3 Open vs Closed Mind Signs

| Aspect | Closed Mind | Open Mind |
|--------|-------------|-----------|
| Response to challenge | Frustration, don't ask why | Curiosity, divergence analysis |
| Error handling | Self-proof > truth | Update risk > ego risk |
| Communication | Assertion > inquiry | Believability weighting |
| Perspective | Assume not understood | Mental simulation of others |
| Truth claim | Rhetorical disclaimer | Contextual appropriateness |
| Dialogue | Monopolize, interrupt | Listen > speak |
| Capacity | Single idea occupancy | Parallel evaluation |
| Learning | No failure-derived growth | Continuous belief update |

### Two-Minute Rule
When disagreeing: let other person speak for 2 min uninterrupted

### Team Synergy
```
OM + OM = decision quality ↑↑
Group synergy >> solo performance
```

## 3.4 How to Become Radically Open-Minded

### Pain-Guided Reflection
- Psychological pain → potential error in held belief
- Physical signs: tension, irritability (amygdala activation)
- Buffer time → calm reflection → higher-self control

### Habit Formation
- Negative emotion = trigger for deliberate thinking
- Consistent practice → direct OM state
- ~18 months to build habit

### Blind Spot Management
- Self-document poor decisions + others' observations
- List critical areas → keep visible
- Never act in blind spots without external check

### Meditation Path
- Transcendental practice → perspective, calmness, creativity

### Evidence Primacy
- Train conscious override of subconscious filtering
- Decision query: "Can you point to clear facts?"

### Social Scaffolding
- Calm discourse prevents amygdala hijacking in others
- Evidence requests > opinion battles

### Tool Augmentation
- Decision systems = emotionless logic + data processing
- Machine thinking adoption → competitive edge

### Terminus Condition
- Defense stops when: evidence consensus > personal view
